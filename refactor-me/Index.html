WebAPI 2.0 Demo application

Using Microsoft.Net Framework 4.5 + WebAPI 2 + Log4Net + EntityFramework 6
Version 1.0
GET: /v1/products - Retrieves all products Returns a json list of all the available products.  Example
GET: /v1/products/?name={ProductName} - Retrieve all products matching the specified name. Returns the specified product details if found otherwise an empty or null value is returned.  Example
GET: /v1/products/{id} - Retrieve the product that matches the specified ID - ID is a GUID. Returns the specified product details if found otherwise an empty or null value is returned.
POST: /v1/products - Add a new product. Returns the product details if added successfully.
PUT: /v1/products/{id} - Update the product that matches the specified ID. Returns the product details if updated successfully.
DELETE: /v1/products/{id} - Delete the product that matches the specified ID and its options. Returns http status code 204 if deleted successfully.
 
Version 2.0
GET: /v2/products - Retrieves all products Returns a json list of all the available products.  Example
GET: /v2/products/?name={ProductName} - Retrieve all products matching the specified name. Returns the specified product details if found otherwise an empty or null value is returned.  Example
GET: /v2/products/{id} - Retrieve the product that matches the specified ID - ID is a GUID. Returns the specified product details if found otherwise an empty or null value is returned.
POST: /v2/products - Add a new product. Returns the product details if added successfully.
PUT: /v2/products/{id} - Update the product that matches the specified ID. Returns the product details if updated successfully.
DELETE: /v2/products/{id} - Delete the product that matches the specified ID and its options. Returns http status code 204 if deleted successfully.
GET: /v2/products/{id}/options - Retrieves all product options Returns the product option details for all the products available.
GET: /v2/products/{id}/options/{optionid} - Retrieve all product options matching a specified product. Returns the product option details for the specified product.
POST: /v2/products/{id}/options - Add a new product option to the specified product. Returns the new product option details for the specified product if successful.
PUT: /v2/products/{id}/options/{optionid} - Update the specified product option for the specified product. Returns the updated product option details for the specified product if successful.
DELETE: /v2/products/{id}/options/{optionid} - Delete the specified product option for the specified product. Returns http status code 204 if deleted successfully.
